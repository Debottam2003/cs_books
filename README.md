# CS Books

Welcome to the Computer Science Books repository! A curated collection of essential resources for mastering fundamental and advanced concepts in computer science.

## Categories

- **Algorithms & Data Structures** – Sorting, searching, graphs, dynamic programming, greedy algorithms
- **System Design** – Scalability, architecture, microservices, distributed systems
- **Operating Systems** – Processes, memory management, concurrency, scheduling
- **Database Management** – SQL, NoSQL, indexing, transactions, query optimization
- **Programming Languages** – Language design, paradigms, best practices, type systems
- **Software Engineering** – Design patterns, testing, code quality, refactoring
- **Networking & Security** – TCP/IP, cryptography, authentication, network protocols
- **Machine Learning & AI** – Neural networks, NLP, computer vision, optimization
- **Web Development** – Frontend frameworks, backend services, APIs, performance
- **Cloud Computing** – AWS, containerization, Kubernetes, serverless architecture

1. Navigate to your category of interest
